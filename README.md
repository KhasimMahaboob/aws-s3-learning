# Amazon S3 Hands-on Project

## 📌 Project Overview

This project demonstrates how Amazon S3 can be used to securely store and manage files using separate public and private buckets.

## 🎯 Objectives

- Create an Amazon S3 bucket
- Understand Bucket vs Object
- Configure private and public buckets
- Upload and manage objects
- Configure an S3 Bucket Policy
- Make selected objects publicly accessible
- Learn secure storage practices

## 🏗️ Architecture

```
                Amazon S3

        ┌─────────────────────┐
        │                     │
        ▼                     ▼
 Private Bucket         Public Bucket
      │                      │
 Personal Docs         Portfolio Assets
 Aadhaar (Private)     Certificates
 Marks Memo            Project Screenshots
```

## 🛠️ AWS Services Used

- Amazon S3
- Bucket Policy
- Object Storage

## 📚 Key Concepts Learned

- Bucket
- Object
- Object URL
- S3 URI
- ARN
- Public Bucket
- Private Bucket
- Bucket Policy
- Block Public Access


## ✅ Outcome

Successfully configured Amazon S3 with separate public and private buckets and controlled access using Bucket Policies.
